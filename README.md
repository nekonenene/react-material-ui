## Installation for deploy

### 1. Clone this repository

### 2. Install dependencies

```sh
brew install nodejs
npm install
```

or

```sh
brew install yarn
yarn install
```

### 3. Run local server

```sh
npm start
```

or

```sh
yarn start
```

Server is located at http://localhost:8013
